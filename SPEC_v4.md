# v4 改造规格书

## 概述
在首页增加「书架」视图，与「笔记」并列切换。书架按分类逐行展示，每本「书」用分类色作背景的卡片样式。

---

## 改动 ①：首页顶部子Tab

**位置**：首页 Tab 内部顶部（搜索框下方、分类 chips 上方）

**Tab 设计**：
- 两个子 Tab 按钮：「📝 笔记」/「📚 书架」
- 选中态：底部有 accent 色下划线，文字加粗
- 切换通过 CSS 显隐对应容器，不重新渲染数据
- 默认选中「笔记」

**布局**：
```
┌─ 🏠 首页 Tab ────────────────────────┐
│  📚 读书笔记    [🔍]                  │
│  ┌──────────┬──────────┐              │
│  │ 📝 笔记  │ 📚 书架  │  ← 子Tab    │
│  └──────────┴──────────┘              │
│  ┌─ 内容区 ──────────────────────┐    │
│  │ 笔记: 列表 + 分类筛选          │    │
│  │ 书架: 按分类分行展示          │    │
│  └────────────────────────────────┘    │
└───────────────────────────────────────┘
```

---

## 改动 ②：书架视图

**整体结构**：
- 按分类分行，每行一个 section
- 每个 section 结构：
  - 左侧 / 顶部：分类标签（带分类色背景 + 白色文字 + 圆角胶囊）
  - 右侧 / 下方：该分类的笔记「书」横向排列
- 笔记多时横滚（横向滑动展示更多）
- 没有笔记的分类不显示

**「书」卡片设计**（核心，要美观）：

```
┌─────────┐
│ ▓▓▓▓▓▓▓ │ ← 书脊（深色阴影）
│         │
│   书名  │ ← 标题（垂直或旋转显示）
│   笔记  │
│         │
│ ▓▓▓▓▓▓▓ │ ← 底部装饰条
└─────────┘
```

**具体参数**：
- 尺寸：宽 110px × 高 160px
- 圆角：6px（书脊 12px 圆角）
- 左侧 6px 深色阴影条作为「书脊」
- 底部 8px 同色加深条作为「书底」
- 标题：
  - 垂直显示（writing-mode: vertical-rl，文字从上到下）
  - 字号 1rem，加粗
  - 颜色：白色或半透明白（与背景对比度足够）
  - 显示完整书名，超出省略
- hover：向上抬起 4px（translateY(-4px)）+ 阴影加深
- 选中态：底部加 3px accent 色横条
- 整体有 box-shadow 体现立体感

**分类配色**（每类用不同色系）：
- 哲学：深紫蓝 `#5d4e8c` + 书脊 `#3d3360`
- 历史：暗红棕 `#8b4513` + 书脊 `#5a2d0d`
- 文学：玫红 `#c44569` + 书脊 `#8a2c4a`
- 科技：青蓝 `#2c7a7b` + 书脊 `#1c5253`
- 其他：烟灰 `#6b7280` + 书脊 `#4a5058`

**布局方式**：
- 每个分类行：左对齐的分类标签 + 横向滚动的书区
- 横向滚动隐藏滚动条（`-webkit-scrollbar: none`）
- 移动端：分类标签在书区上方（垂直布局）
- 桌面端：分类标签在左侧（水平布局）

**交互**：
- 点击书 → 加载该笔记预览（与笔记 Tab 一致）
- 长按/hover 弹 tooltip 显示笔记元信息（日期、标签）

## 改动 ③：状态管理

**新增**：
- `STATE.homeView` = 'list' | 'shelf'
- 默认 'list'
- 切换 Tab 时只切换显隐容器，不重渲染
- 书架容器和笔记容器互斥显示

**localStorage 持久化**：
- `notes-home-view` 键，记住用户上次选择

---

## 技术要求
- 单文件 index.html 修改
- 不引入新依赖
- 保持现有暖色调主题、暗色模式、safe-area
- 书脊阴影用 box-shadow 或 gradient 实现
- 移动端触摸滑动支持（CSS `-webkit-overflow-scrolling: touch`）
- 完成全局一致性审查
