01中国金融学派的标杆教科书

由"中国金融学之父"黄达先生主编的《金融学》，是中国人民大学出版社"21 世纪金融学系列教材"的旗舰之作；自 2003 年初版以来历经多版修订，现行通用第六版由黄达、张杰共同主编，在保留宏观金融理论根基的同时，吸纳了利率市场化、汇率市场化、金融科技与开放金融体系的最新进展。

编写定位

  • 面向高年级本科与硕士：是央行、银保监、研究院、高校研究生入学考试的核心参考

章节结构

  • 第一篇 范畴：货币、信用、利息率、外汇与汇率

Chapter I · 第一篇 范畴
01

货币 · 信用· 利率

金融学最底层的三个范畴：什么是货币、信用如何运行、利率如何形成与决定。这三块是后续宏观分析、市场分析、政策分析的共同基石。

Three Pillars · 三大基础范畴

02货币的本质与信用的形态

教材在开篇就明确：货币与信用并不是孤立的"工具"，而是经济运行的两条主血管。把握这两条主线，才能理解后续的市场、机构与政策。

01 · 货币职能

货币的五大职能

价值尺度 · 流通手段 · 贮藏手段 · 支付手段 · 世界货币。教材延续马克思货币职能体系，并融合现代货币理论，强调"流通手段+支付手段"是现代信用货币的核心。

02 · 货币制度

货币制度的演变

从实物货币 → 金属货币（金本位 / 复本位）→ 不兑现的信用货币 → 电子货币 / 数字货币。重点理解 1971 年布雷顿森林体系瓦解后，全球进入纯信用货币时代的逻辑。

03 · 信用形式

五大信用形式

商业信用 · 银行信用 · 国家信用 · 消费信用 · 国际信用。其中银行信用是现代经济的主导形式；国家信用通过国债将财政与货币联结，是宏观调控的关键支点。

Interest & Rates · 利息与利率

03利息率：资金的价格信号

利率是资金借贷市场的均衡价格，也是连接金融与实体经济最关键的传导变量。教材从三个层次展开：利率的种类、利率的决定理论、利率的期限与风险结构。

  • 种类：名义/实际、固定/浮动、市场/官定、基准/差别利率（LPR 是中国当前的核心基准）
  • 决定理论：古典储蓄投资论、可贷资金论、流动性偏好论（凯恩斯）、IS-LM 综合论
  • 期限结构：预期理论、市场分割理论、流动性溢价理论 —— 解释收益率曲线为何上斜、倒挂
  • 风险结构：违约风险、流动性、税收差异共同决定不同信用主体之间的利差
  • 利率市场化：中国 1996 年启动至 2015 年存款利率上限放开，2019 年 LPR 改革标志正式形成

i ≈ r + πe
名义利率 ≈ 实际利率 + 预期通货膨胀率。预期通胀上行时，名义利率必须同步上行才能保持实际利率稳定。

Chapter II · 第二篇 机构与市场
02

金融市场· 金融机构

资金从盈余者流向赤字者的两条主路径：直接金融市场与间接金融机构。教材在此框架下系统梳理货币市场、资本市场、外汇市场，以及银行体系的层级结构。

Markets · 三大金融市场

04货币 · 资本 · 外汇与衍生品市场

按交易工具的期限和性质，金融市场划分为短期资金的货币市场、长期融资的资本市场，以及跨境资金交易的外汇与衍生品市场。三者互为补充，共同形成完整的金融体系。

01 · Money Market

货币市场

期限1 年以内。子市场包括同业拆借、回购、票据贴现、大额存单、短期国债。核心功能：流动性管理与基准利率形成（DR007、SHIBOR）。

02 · Capital Market

资本市场

期限1 年以上。包括股票市场（主板/创业板/科创板/北交所）、债券市场（利率债/信用债）、长期信贷。核心功能：长期资本形成与价格发现。

03 · FX & Derivatives

外汇与衍生品

包括即期、远期、掉期、期权、期货、利率互换、CDS。核心功能：跨境资金调剂、汇率发现、风险对冲。中国 2015 年 8.11 汇改后人民币兑美元逐步进入双向波动机制。

Central Bank · 中央银行

06中央银行：发钞银行 · 银行的银行 · 政府的银行

中央银行处于现代金融体系的顶层。教材从起源、性质、职能、独立性四个维度展开，强调央行不是普通银行，其负债（基础货币）是整个信用货币体系的基石。

  • 三大基本职能：发钞银行（垄断货币发行）/ 银行的银行（最后贷款人 + 清算中心 + 集中存款准备金）/ 政府的银行（代理国库 + 经理国债）
  • 资产负债表：资产侧 = 国外资产 / 对政府债权 / 对存款机构债权 / 对其他金融机构债权；负债侧 = 储备货币（流通中现金 + 金融机构存款）+ 政府存款 + 央行票据
  • 独立性：目标独立 / 工具独立 / 人事独立 —— 越独立，越能抵御短期政治压力下的通胀偏向
  • 中国实践：1984 年人行专门行使央行职能；2003 年银监会从人行分立；2023 年金融监管总局成立后，人行更聚焦货币政策与宏观审慎"双支柱"
基础货币 · MB
B = C + R
基础货币 = 流通中现金（C） + 银行准备金（R）。央行通过控制 B 的总量，间接影响整个货币供给体系。
Asset Pricing · 资产定价基础

07四个最基本的定价工具

教材在金融市场篇中嵌入了"现值 → 股票 → 债券 → 风险定价"四级递进的资产定价工具箱。这是把金融市场理论与实务连接起来的最小公式集。

01 · 现值 PV
PV = ∑ CFt / (1+r)t
所有定价的最底层逻辑：未来现金流按贴现率 r 折回今日。r 反映时间价值与风险溢价。
02 · 股票 DDM / Gordon
P0 = D1 / (k − g)
股利折现模型常见简化形式（增长率 g 恒定且 g < k）。k 是必要回报率，g 是股利永续增长率。
03 · 债券 · 久期 D
D = ∑ t · wt　　ΔP/P ≈ −D · Δy
麦考利久期是各期现金流时间的加权平均。修正久期度量利率敏感性：利率上行 1bp，债券价格约下跌 D bp。
04 · 风险定价 CAPM
E(Ri) = Rf + βi · [E(Rm) − Rf]
资本资产定价模型：单个资产的预期收益由无风险利率 Rf 与系统性风险 β 的市场溢价共同决定。β 是相对市场组合的风险敞口。

教材在第二篇 · 金融市场章中给出推导，建议结合米什金《货币金融学》对应章节交叉印证。

03

货币需求 · 供给· 政策传导

从"为什么持有货币"到"货币如何被创造"，再到"货币政策如何影响产出与物价"。这一篇把宏观货币的供求与政策连接成完整闭环，是教材的理论核心区。

Money Demand · 货币需求理论

08三大学派的货币需求函数

货币需求理论回答的是"经济主体为什么、按多少比例、把财富以货币形式持有"。教材按时间脉络梳理古典 → 凯恩斯 → 弗里德曼三大流派。

01 · Classical

古典学派 · 现金交易方程

MV = PY

费雪方程式。V（流通速度）和 Y（产出）短期稳定，因此货币 M 的变化直接影响物价 P —— 货币数量论的经典结论。剑桥方程 M = kPY 与之等价。

02 · Keynes

凯恩斯 · 流动性偏好

L = L₁(Y) + L₂(i)

持有货币的三大动机：交易、预防、投机。前两者随收入正相关，投机性需求与利率反向。利率足够低时进入"流动性陷阱"，货币政策失效。

03 · Friedman

弗里德曼 · 现代货币数量论

Md/P = f(Yp, ...)

把货币视为一种资产，需求取决于恒久收入 Yp、各类资产收益率、偏好。结论：货币需求函数稳定，央行盯住 M 的增速即可稳定经济 —— 货币主义的政策主张。

Money Supply · 货币供给

09货币层次与货币乘数

现代信用货币是分层次的。央行直接控制基础货币（MB），通过商业银行的存款派生过程，逐级形成 M0 → M1 → M2 的总量结构。

M0
现金流通中
流通中的现金。流动性最强、与民生最直接相关。
M1
M0 + 活期
狭义货币 = M0 + 单位活期存款。代表交易性货币，反映企业经营活跃度。
M2
M1 + 准货币
广义货币 = M1 + 居民储蓄 + 单位定期存款 + 其他存款。央行调控的主要中介目标之一。
m
M / B乘数
货币乘数 m = (1+c)/(rd+re+c)，其中 c 为现金比率，rd 法定准备金率，re 超额准备金率。

注：中国央行 M3 口径已较少使用；社会融资规模（社融）作为辅助指标反映实体经济从全社会获得的资金总量。

Quant · 货币计量公式集

10三组核心公式：从基础货币到 IS-LM

① 货币乘数完整推导

M = C + D ;　B = C + R
⇒ m = M/B = (C+D) / (C+R)
上下同除 D，令 c = C/D，r = R/D：
m = (1 + c) / (rd + re + c) c：现金/存款比率　rd：法定准备金率　re：超额准备金率

乘数 m 受三个比率共同影响：央行调 rd 是直接控制；re 受银行风险偏好与流动性压力影响；c 受居民支付习惯（现金 vs 电子货币）影响。

② Fisher 方程与实际利率

(1+i) = (1+r)(1+πe)
展开后近似：　i ≈ r + πe i：名义利率　r：实际利率　πe：预期通胀率

考点延伸：当 πe 上行而 i 不变 → 实际利率 r 下行 → 货币政策被动宽松。这是"通胀税"的核心机制。

③ IS-LM 联立方程

IS　Y = C(Y−T) + I(r) + G + NX
⇒ 商品市场均衡，r↑ ⟹ Y↓

LM　M/P = L(Y, r)
⇒ 货币市场均衡，Y↑ ⟹ r↑ 两条曲线交点同时确定均衡产出 Y* 与利率 r*
扩张性财政　G↑
IS 右移 → Y↑ r↑（"挤出效应"使乘数变小）
扩张性货币　M↑
LM 右移 → Y↑ r↓（流动性陷阱时 LM 水平段失效）
总需求　AD
IS-LM 进一步导出 AD 曲线，与 AS 联立得 P*、Y*

④ 货币流通速度 V

MV = PY ⇒ V = PY / M
ΔM/M + ΔV/V = ΔP/P + ΔY/Y 费雪方程的增长率形式 —— 中国 2010s 后 V 持续下行，使 M2 高增长未必转化为高通胀

Inflation & Deflation · 物价失衡

12通货膨胀与通货紧缩对照

I · 通胀成因

需求拉上 / 成本推动 / 结构性 / 输入型

过度需求驱动总需求曲线右移；工资—成本螺旋；部门发展不平衡；进口商品价格上升通过汇率渠道传入。

  • 财政赤字货币化是历史上恶性通胀的主因
II · 通胀影响

财富再分配 · 经济效率损失 · 预期紊乱

从债权人转向债务人、从固定收入者转向资产持有者；菜单成本与皮鞋成本；预期一旦形成便自我强化。

  • 恶性通胀（年率 > 50%）会摧毁货币职能
III · 通缩成因

有效需求不足 · 资产负债表衰退 · 输入型

持续物价下行 + 货币供应紧缩 + 经济衰退三者叠加。一旦陷入"债务—通缩"螺旋，治理难度远高于通胀。

  • 日本"失去的三十年"是通缩的典型案例
IV · 治理工具

紧缩 vs 宽松 · 货币 vs 财政 · 预期管理

反通胀：紧货币 + 紧财政 + 收入政策 + 预期引导；反通缩：宽松货币 + 积极财政 + 结构改革 + 必要时进入零利率/负利率。

  • 非常规货币政策（QE / YCC / 前瞻指引）已成为应对通缩的标准工具箱
Regulation · 监管演进

13中国金融监管框架四十年

1984

央行职能分立

中国人民银行剥离工商信贷业务，专门行使中央银行职能；工商银行成立。中国金融体系开始由"大一统"向二元结构转型。

1995

《商业银行法》《中国人民银行法》

首次以基本法律形式确立商业银行的市场化运营原则与人行央行地位，金融立法体系从此建立。

1998 · 2003

"一行三会"分业监管成形

1998 年保监会成立；2003 年银监会从人行剥离 —— 形成 "人行 + 银监会 + 证监会 + 保监会" 的分业监管格局。

2017 – 2018

金融稳定发展委员会 + 银保合并

设立国务院金融稳定发展委员会统筹协调；银监会、保监会合并为"银保监会"，监管走向功能监管与综合监管。

2023

党和国家机构改革：在银保监会基础上组建金融监管总局，把人行的金融控股公司监管职能、证监会的投资者保护职能纳入；形成"一行（人行）+ 一总局 + 一会（证监会）"的新格局。

Open Economy · 开放经济

14国际收支 · 汇率制度 · 资本流动

开放经济条件下，国内货币政策必须在"汇率稳定 / 货币政策独立 / 资本自由流动"的不可能三角中作权衡。教材在此章把开放金融的三大议题串成一条逻辑链。

01 · BOP

国际收支平衡表

由经常账户、资本与金融账户、储备资产三大类构成。经常账户反映实体竞争力；资本账户反映金融一体化深度。失衡可通过本币贬值、外汇干预、资本管制来再平衡。

02 · FX Regime

汇率制度光谱

从硬钉住（货币局/共同货币）→ 软钉住（爬行钉住、目标区）→ 浮动（管理浮动、自由浮动）。中国 2005 年汇改后实行"以市场供求为基础、参考一篮子货币、有管理的浮动汇率制"。

03 · Capital Flow

资本流动与三元悖论

蒙代尔三元悖论：汇率稳定 / 资本自由流动 / 货币政策独立三者不可兼得，只能选两项。中国当前选择"货币政策独立 + 部分资本管制 + 有管理的浮动"组合。

Parity · 国际金融三大平价

15利率平价 · 汇率平价 · 不可能三角的数学化

开放经济中"三大平价"是教材最高密度的公式区。它们把利率、通胀、汇率、跨境套利串成统一方程，也是央行理解外汇市场的核心工具。

01 · 抛补利率平价 CIP
F / S = (1 + i本) / (1 + i外)
远期汇率 F 与即期汇率 S 的比值由两国利差决定。CIP 通过远期合约锁汇套利，若不成立即存在无风险套利空间，在自由资本流动下几乎严格成立。
02 · 非抛补利率平价 UIP
E(St+1) / St = (1 + i本) / (1 + i外)
不锁汇情形下，预期汇率变动 ≈ 利差。高息货币会被预期未来贬值以抵消利差。"高息货币持续走强"现象就是 UIP 在现实中的偏离（carry trade）。
03 · 购买力平价 PPP
绝对：S = P本 / P外　相对：ΔS/S ≈ π本 − π外
一价定律的国际化。绝对 PPP 在短期几乎不成立；相对 PPP 在长期、可贸易品上有较强解释力。中国 1994 年汇率并轨后人民币实际有效汇率走势可由相对 PPP 大体解释。
04 · 不可能三角 · 数学化
∂Y/∂G | 固定汇率 + 资本流动 = 0
Mundell-Fleming 模型证明：在固定汇率 + 完全资本流动下，财政政策完全有效、货币政策完全无效；浮动汇率下结论反转。三元悖论的数学根源就在这里。

PPP 通常给出长期均衡；UIP 给出短期预期路径；CIP 给出无风险套利下界。三者构成开放经济金融分析的"三角坐标系"。
