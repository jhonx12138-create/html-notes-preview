⬇️L5 依赖 L4 模型推理
L4 模型层 Transformer · 条件概率预测器
① Transformer 架构核心
📐 Block 数据流
输入 → 层归一化 → 多头自注意力 → 残差连接(+) → 层归一化 → 前馈网络(FFN) → 残差连接(+) → 输出 （重复 N 层堆叠）
Q/K/V 注意力查询/键/值：动态计算词间相关性，提取上下文信息 多头注意力多组QKV并行，从不同语义空间捕捉关系 位置编码给每个词注入位置信号，弥补注意力机制的乱序缺陷 残差连接输入原样加到输出上，防止深层网络梯度消失 前馈网络 FFN对每个位置独立进行深度信息加工的全连接层
② 训练机制
📈 核心训练要素
损失函数 (交叉熵)衡量预测词与真实词差异，训练目标是最小化它 反向传播 + 梯度下降从输出向输入逐层计算误差梯度，更新参数 优化器 (Adam/AdamW)自适应调整学习率的优化算法 学习率调度 分布式训练将模型和数据切分到多张GPU上协同训练
🔄 训练流程
预训练海量文本上做填空题，获得基础语言能力 SFT 监督微调用高质量问答对教会模型遵循指令 RLHF / DPO 对齐依据人类偏好优化，让模型更安全、更有帮助
③ 多模态与 Token 化
Tokenizer 分词器将文本切成最小单元(Token)并映射为数字，是模型输入基础 图像 Token 语音 Token 视频 Token 跨模态对齐让文字、图像等不同模态在向量空间中语义一致 统一序列建模
④ 关键超参
上下文窗口模型一次能处理的最大Token数，超出部分会遗忘 温度 Temperature控制随机性：低则确定保守，高则创意奔放 Top-P / Top-K
⑤ 模型家族
Decoder-Only (GPT, LLaMA)自回归逐词生成，擅长对话与文本续写 Encoder-Only (BERT)双向理解上下文，适合分类、抽取等理解任务 Encoder-Decoder (T5, Whisper)输入编码→输出解码，适合翻译、语音识别 Diffusion 扩散模型从噪声逐步去噪生成图像/视频，Stable Diffusion为代表
