Section 01

CSS 是造型师，HTML 是模特

CSS（层叠样式表）专门告诉浏览器"怎么画"。HTML 只管“有什么”，CSS 才管“好不好看”。它们是永远搭档的两个人。

📄

写在单独文件里

新建 style.css，在 HTML 里用 <link> 引入。

✅ 推荐做法
<link rel="stylesheet" href="style.css">
🏷️

写在 <style> 标签里

直接放在 HTML 的 <head> 里，不用额外文件。

⚡ 快速测试用
<style> h1 { color: red; } </style>
✍️

直接写在标签上

用 style="..." 属性，直接给某个元素设样式。

⚠️ 不建议滥用
<p style="color:red"> 文字 </p>
Section 02

一句 CSS 由三个部分组成

h1 {
  color: purple;
}
选择器
"要打扮谁"
属性
"打扮什么"
值
"打扮成啥样"

💡 每条声明末尾都有分号 ;，别忘了

三种最常用的选择器

标签名
选中页面上所有这个标签
比如所有 h1 都变红
h1 { color: red; }
.class 名
选中带这个 class 的元素
精准点名，推荐用这个
.card { ... }
#id 名
选中这个唯一 id 的元素
全页面只有一个目标
#logo { ... }
Section 03

先记住这 9 个属性就能玩转样式

🎨 color

文字颜色

color: red;
color: #7c4dff;
颜色名或十六进制码

🖼️ background

背景颜色/图片

background: pink;
background: #fff;
也可以放渐变或图片

🔤 font-size

字号大小

font-size: 16px;
font-size: 1.5rem;
px 是像素，最直观

💪 font-weight

文字粗细

font-weight: bold;
font-weight: 700;
400 = 正常，700 = 粗

📐 padding

内边距

padding: 20px;
内容和边框之间的空白
像给礼物盒子加泡沫

🏕️ margin

外边距

margin: 16px auto;
元素和元素之间的距离
auto 可以实现居中

🔲 border-radius

圆角

border-radius: 12px;
数值越大越圆
50% = 变成圆形

🌫️ box-shadow

投影

box-shadow: 0 4px 12px rgba(0,0,0,.15);
让卡片"浮"起来

📏 width / height

宽 / 高

width: 300px;
width: 100%;
% 是相对父元素的比例

Section 04 · Box Model

每个元素都是一个嵌套的盒子

这是 CSS 最重要的概念。看懂这张图，布局就不再玄学了。

Margin · 外边距
Border · 边框
Padding · 内边距
Content · 内容
你好

Margin — 外边距

元素和其它元素之间的距离。就像两栋楼之间的间隔。

margin: 上 右 下 左;

Border — 边框

元素的外框线。可以设粗细、颜色、实线/虚线。

border: 2px solid #ccc;

Padding — 内边距

内容和边框之间的呼吸空间。让文字不贴着边框，像礼盒里的泡沫。

padding: 16px 24px;
