04

Transformer 自注意力机制演示

输入句子："我 很 喜欢 苹果，我 爱 吃"  →  预测下一个词

计算步骤

  1. Token化：将输入文本拆分为 token 序列
  2. 嵌入：每个 token 转为向量（如 512 维）
  3. 位置编码：加入位置信息
  4. 多头注意力计算：每个头独立计算 Query、Key、Value → Q·KT → Softmax → 加权求和
  5. 多个头并联：头 1 关注语法、头 2 关注语义、头 3 关注位置邻近 …… 拼接后线性变换
  6. 前馈网络 + 残差连接 + 层归一化
  7. 输出：下一个词的概率分布（蛋糕 0.4，香蕉 0.3 ……）
05

当前架构存在的核心问题

⚠️ 问题说明
计算复杂度高自注意力机制需要计算每个 token 与其他所有 token 的关系，复杂度随序列长度平方增长
上下文长度限制受显存、位置编码限制，以及注意力稀释问题，上下文窗口有限
可解释性差只能看到输入输出，看不到模型决策过程，也无法理解高维空间的语义
AI 幻觉模型解决的是“概率最优”，而非“事实最优”。它学会的是“什么样的文本看起来合理”，而不是“什么是真的”

🧠 Transformer 架构下 LLM 的本质：基于贝叶斯推断（条件概率）的逐个 token 预测

✨ 为什么这么强 —— 涌现性：模型在规模（数据、参数、计算量）跨越某个临界点后，突然获得了其组成部分或小模型中未曾出现、也未被明确训练过的能力，比如思维链、上下文理解、代码编辑……

AI 发展的三大流派

流派核心思想代表技术当前地位
符号主义符号逻辑推理专家系统、知识图谱辅助 LLM（事实校验）
连接主义神经元网络学习深度学习、Transformer★ 绝对主流
行为主义环境交互 + 奖惩强化学习（RLHF）重要补充（微调 / 决策）

💡 现代 LLM = 连接主义为主 + 符号主义（思维链逻辑）+ 行为主义（RLHF）
