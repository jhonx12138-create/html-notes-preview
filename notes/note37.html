Section 01

三人组各司其职，JS 是那双手

一个网页由三层构成，缺一不可。把它想成一个人：

🦴
HTML

骨架

定义"页面上有什么"：标题、段落、按钮、图片。没有它，什么都没有。

👗
CSS

衣服和化妆

决定"长什么样"：颜色、字体、圆角、卡片阴影。好看不好看，全靠它。

🙌
JavaScript

动作和大脑 ← 今天

决定"能干什么"：点了按钮变色、输入关键词搜索、自动轮播图片。这就是 JS。

💡 小记忆法：HTML = 名词（这是什么），CSS = 形容词（长什么样），JavaScript = 动词（能干什么）。

Section 02

JS 的四个基本积木

先认识这四个，剩下的全是它们的组合。

📦
积木 1

变量 — 存东西的盒子

// 声明一个变量，存你的名字 let name = "小明"; let age = 18; let isStudent = true; // 用的时候直接写变量名 console.log(name); // → "小明"

三种存法：let（可改）、const（不可改）、var（老写法，少用）

⚙️
积木 2

函数 — 打包好的一套动作

// 定义一个打招呼的函数 function greet(name) { alert("你好，" + name); } // 调用它 greet("小明"); // 弹出"你好，小明"
🔀
积木 3

条件 — 根据情况选择

let score = 85; if (score >= 60) { console.log("及格了 🎉"); } else { console.log("再努力一下 💪"); } // → 及格了 🎉
🔄
积木 4

循环 — 重复做同一件事

// 打印 1 到 5，不用复制粘贴 5 遍 for (let i = 1; i <= 5; i++) { console.log("第" + i + "次"); } // → 第1次 第2次 ... 第5次
Section 03 · DOM

JS 用 DOM 来操控 HTML

DOM（文档对象模型）听起来很技术，但意思很简单：浏览器把整个 HTML 页面变成一棵"树"，JS 可以通过这棵树找到任何元素，然后改它。

① 找到元素

// 找到 id 叫 "title" 的那个元素 const el = document.getElementById("title"); // 找到第一个 class 叫 "card" 的元素 const card = document.querySelector(".card"); // 找到所有 p 标签 const ps = document.querySelectorAll("p");

② 改掉它

// 改文字 el.textContent = "新标题"; // 改 CSS 样式 el.style.color = "red"; // 加一个 class el.classList.add("active"); // 隐藏它 el.style.display = "none";

Section 04 · Events

事件：JS 在监听你的每一个动作

用户点击、滑动、输入、移动鼠标——每个动作都是一个"事件"。JS 通过 addEventListener 来监听，然后做出反应。

// HTML 里有一个按钮 <button id="myBtn">点我</button> <p id="msg"></p> // JS 监听这个按钮的点击事件 const btn = document.getElementById("myBtn"); btn.addEventListener("click", function() { document.getElementById("msg").textContent = "你点了我！🎉"; });

Section 05 · 真的能用的小功能

动手：下面两个你可以直接用

▼ 功能一：点击显示一句话
等待你的点击...
const messages = [ "你好呀！👋", "再点一下呢？😄", "就知道你停不下来 😂" ]; let count = 0; document.getElementById("demo-btn") .addEventListener("click", function() { document.getElementById("demo-output").textContent = messages[count % messages.length]; count++; });
▼ 功能二：加减计数器
0
let n = 0; const display = document.getElementById("demo-counter-display"); document.getElementById("plus-btn") .addEventListener("click", function() { display.textContent = ++n; }); document.getElementById("minus-btn") .addEventListener("click", function() { display.textContent = --n; });
