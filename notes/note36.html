Chapter 1 · Fundamentals

HTML 是什么

HTML（HyperText Markup Language）是构建网页的标准标记语言，不是编程语言，而是结构描述语言——用标签告诉浏览器"这是标题""这是段落""这是图片"。

一个最小 HTML 文件的四个必要骨架：

<!-- 1. 文档类型声明 --> <!DOCTYPE html> <!-- 2. 根元素 --> <html lang="zh-CN"> <!-- 3. 头部元信息（不可见） --> <head> <meta charset="UTF-8"> <title>页面标题</title> </head> <!-- 4. 主体内容（可见） --> <body> <h1>Hello, World!</h1> </body> </html>
现行标准
Living
Standard
由 WHATWG 持续维护
不再有"版本号"
<!DOCTYPE html>
告知浏览器使用 HTML5 标准渲染模式
<html>

<head>

<body>
所有可见内容的容器
Chapter 1 · History

HTML 的演化历程

1991

HTML 1.0

Tim Berners-Lee 在 CERN 发明，最初只有 18 个标签，仅支持文本与链接

1995

HTML 2.0

IETF 发布首个正式规范，引入表单（form / input）

1997

HTML 3.2 / 4.01

W3C 接手，引入表格、CSS 挂载；4.01 成为稳定基石

2000

XHTML 1.0

基于 XML 的严格版本，要求所有标签小写、闭合，属性加引号

2014

HTML5

WHATWG + W3C 联合发布，引入语义标签、音视频、Canvas、Web API

今天

Living Standard

WHATWG 持续迭代，无版本号，浏览器厂商实时跟进，这就是现在的 HTML

Chapter 2 · Syntax

元素的三种形态

所有 HTML 元素按渲染行为分为三类，理解它们是掌握布局的第一步。

📦

块级元素

Block

独占一行，宽度默认撑满父容器。代表：div p h1-h6 ul section。可设置宽高。

📝

内联元素

Inline

随文字流排列，不独占行，宽高由内容决定。代表：span a strong em img。

🔲

空元素

Void

没有内容也没有闭合标签，自闭合。代表：<br> <hr> <img> <input> <meta>。

<!-- 元素解剖：开始标签 + 内容 + 结束标签 --> <p class="intro" id="first">这是一段文字</p> <!-- 空元素自闭合，HTML5 中结尾斜杠可选 --> <img src="photo.jpg" alt="描述文字">
Chapter 2 · Rules

属性、嵌套、注释与布尔值

📌 属性写法

<!-- 标准属性：name="value" --> <a href="https://example.com" target="_blank" rel="noopener">链接</a> <!-- 布尔属性：存在即为 true --> <input type="checkbox" checked disabled> <!-- data-* 自定义属性 --> <div data-user-id="42">用户</div>

🔗 嵌套规则

<!-- ✅ 正确：先开后闭，对称嵌套 --> <ul> <li><a href="#">条目</a></li> </ul> <!-- ❌ 错误：交叉嵌套 --> <b><i>文字</b></i> <!-- 注释语法 --> <!-- 这是注释，不会被渲染 -->
全局属性
所有元素都支持：id、class、style、lang、title、data-*、hidden、tabindex

Chapter 3 · Deep Dive

三大核心系统

📋

表单系统

<form> · <input> · <select>
<form action="/submit" method="post"> <label for="name">姓名</label> <input id="name" type="text" required placeholder="请输入"> <button type="submit">提交</button> </form>

input type 支持 text/email/password/number/date/file/checkbox/radio 等 22 种类型

🏗️

语义化结构

<main> · <article> · <nav>
<body> <header>导航栏</header> <main> <article>主文章</article> <aside>侧边栏</aside> </main> <footer>页脚</footer> </body>

语义标签帮助搜索引擎理解内容，提升 SEO 与可访问性（屏幕阅读器）

🎬

多媒体嵌入

<video> · <audio> · <iframe>
<video controls width="640"> <source src="film.mp4" type="video/mp4"> <source src="film.webm" type="video/webm"> </video> <!-- figure + figcaption 图文组合 --> <figure> <img src="chart.png" alt="季度图表"> <figcaption>2026 Q1 数据</figcaption> </figure>

Chapter 4 · Practice

写一个完整 HTML 页面的 6 步流程

1

声明文档类型

第一行写 <!DOCTYPE html>，开启标准渲染模式，避免浏览器进入 Quirks 模式

2

填写元信息 <head>

charset、viewport、title、description、og:* 等——影响 SEO、分享预览和移动端适配

3

搭建语义骨架

用 header / main / footer 划区，用 section / article / aside 细分，避免 div 地狱

4

填充内容

标题用 h1-h6，文本用 p，列表用 ul/ol，表格用 table，图片记得写 alt

5

挂载样式与脚本

CSS 在 <head> 里用 <link rel="stylesheet"> 引入；JS 在 </body> 前用 <script> 引入，或加 defer/async

6

验证与检查

用 validator.w3.org 做 HTML 合规验证；用 Lighthouse 检查可访问性与性能评分

<!-- head 元信息完整示例 --> <head> <meta charset="UTF-8"> <meta name="viewport" content="width=device-width, initial-scale=1.0"> <meta name="description" content="页面描述（SEO）"> <title>页面标题 · 网站名称</title> <link rel="stylesheet" href="style.css"> </head>
