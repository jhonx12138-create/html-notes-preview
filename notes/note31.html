🚀 Agent 框架类（Harness）的发展方向

正在发生
🔌

MCP 协议标准化

像 USB-C 统一外设，Agent 可动态发现并调用兼容工具。

🤝

多 Agent 协作

多个 Agent 分工完成复杂项目，形成团队化工作流。

🧠

长期记忆 + RAG

跨会话记住偏好与经验，结合向量检索实现学习与成长。

💻

代码生成 + 沙箱

现场写代码解决新问题，在安全沙箱中扩展能力边界。

🛡️

安全护栏完善

多层输入输出过滤，防止毁灭性操作。

👤

人机协同深化

高风险操作自动暂停等待批准，监督-执行协作模式。
