# v5 改造规格书

## 改动 ①：笔记列表右滑操作

### 移除内容
- 首页「📝笔记 / 📚书架」子Tab 移除，只保留笔记列表视图
- 书架入口保留在首页其他地方（或暂时隐藏，改为长按入口）

### 右滑机制
- 每条笔记项支持向左滑动（手指从右往左划）
- 滑动后露出操作按钮区（固定在右侧，宽度约 120px）
- 操作按钮：**修改**（蓝色）/ **删除**（红色）/ **置顶**（橙色）
- 滑动距离 < 60px 时自动弹回
- 滑动距离 ≥ 60px 时吸附展开，露出按钮
- 点击笔记项空白区域 → 收起按钮（若有展开）
- 点击其他笔记 → 前一个展开的自动收起
- iOS/Android 触摸事件支持（touchstart/touchmove/touchend + pointer 事件）

### 按钮功能
- **修改**：弹出修改弹窗（modal），表单含：
  - 文件名（可编辑 input）
  - 分类（下拉选择，同上传页）
  - 标签（逗号分隔 input）
  - 保存 / 取消按钮
  - 保存后同步更新 IndexedDB（用户笔记）或内存（内置笔记）
- **删除**：调出现有确认弹窗 `showConfirm`，确认后删除
- **置顶/取消置顶**：
  - 未置顶笔记 → 点击置顶 → 笔记移到列表顶部，按钮变为「取消置顶」
  - 已置顶笔记 → 点击取消 → 恢复原位置
  - 置顶状态存 `localStorage` 键 `notes-pinned`（数组，存 noteId）
  - 置顶笔记在列表顶部，之间有分割线或视觉区分

### 视觉
- 按钮区背景：修改 `#2196F3` / 删除 `#f44336` / 置顶 `#FF9800`
- 白色图标 + 文字（小于等于3个字）
- 按钮区圆角，与笔记项高度一致
- 滑动动画使用 CSS transition

## 改动 ②：书架长按操作

### 交互
- 书架中的书卡片支持长按（800ms）
- 长按后弹出操作菜单（action sheet 风格，底部弹出）
- iOS 风格：底部白色圆角 sheet + 选项列表 + 取消
- 选项：
  - 📝 修改信息
  - 🗑️ 删除笔记（仅用户笔记）
  - 置顶/取消置顶
  - 取消

### 功能
- 修改 / 删除 / 置顶 功能与改动①一致
- 操作完成后自动刷新书架

### 实现
- touchstart + setTimeout 800ms → 触发长按
- touchmove 超过 10px → 取消长按
- touchend 在 800ms 前 → 取消长按
- 长按触发时设备震动反馈（navigator.vibrate(20)）

## 改动 ③：书架书卡片排版优化

### 当前问题
- 竖排文字只能显示 4-5 个字，书名长会截断

### 改为横排卡片
- 卡片尺寸：宽度自适应（最小 140px）× 高度 auto
- 布局改为横排显示：
```
┌──────────────┐
│ ▓▓▓▓▓▓▓▓▓▓▓▓ │ ← 顶部色条（书标题背景区）
│   书名全称   │ ← 标题横排，可换行，最多3行
│ (完整展示)   │
│              │
│ 📅 2024-01-15│ ← 日期
│ #文学 #哲学  │ ← 标签 chips
│ ▓▓▓▓▓▓▓▓▓▓▓▓ │ ← 底部色条
└──────────────┘
```
- 圆角 10px，带阴影
- 背景仍用分类色系的浅色版（如哲学用浅紫 `#e8e0f0`）
- 顶部和底部保留色条装饰
- 标题字体 bold，黑色/深色文字（不是白色了）
- 标签用彩色小圆点 + 文字
- 整卡 hover 上抬 4px
- 标签文字长的用省略号

### 配色改浅色：
之前用深色系做书封面，现在改为浅色背景 + 深色文字：
- 哲学：背景 `#f0ecfa` / 色条 `#5d4e8c`
- 历史：背景 `#faf0e6` / 色条 `#8b4513`
- 文学：背景 `#fde8ee` / 色条 `#c44569`
- 科技：背景 `#e6f5f5` / 色条 `#2c7a7b`
- 其他：背景 `#f3f4f6` / 色条 `#6b7280`
